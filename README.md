## 🎯 Strategy Tips

### Playing the Classic 3×3 Board

**Against Easy AI:**
- Very beatable - the AI makes completely random moves
- You should win almost every game
- Use this to learn the game and practice

**Against Medium AI:**
- Mix of smart and random strategies (50/50)
- Watch for patterns in opponent's moves
- Most games are winnable with good play
- Try to control the center first

**Against Hard AI:**
- Unbeatable when it plays optimally
- Best realistic outcome: **force a draw**
- Strategy to draw against Hard:
  1. Take center (5) if AI doesn't
  2. Take opposite corners when possible
  3. Create multiple threats simultaneously
  4. Avoid putting yourself in losing positions

**General Winning Strategies:**
- **Center Control:** Middle square is crucial
- **Multiple Threats:** Create situations where you can win multiple ways
- **Block Threats:** Stop opponent from completing 3 in a row
- **Corner Play:** Corners offer more winning possibilities than edges

### Competing on Larger Boards (4×4, 5×5)

- More complex gameplay with 4 or 5 in a row needed
- Center control becomes even more important
- Hard AI becomes even more unbeatable
- Great for experienced players wanting a challenge

## Technical Details

### GUI Implementation

The graphical version uses **tkinter** (Python's standard GUI library) with:
- Object-oriented design with a main `TicTacToeGUI` class
- Event-driven programming for button clicks
- Custom color scheme for better visual appeal
- Delayed AI moves for smooth gameplay experience
- State management for different game screens

### AI Implementation

The AI uses the **Minimax algorithm**, a decision-making algorithm that:
- Explores all possible future game states
- Assumes both players play optimally
- Chooses moves that maximize AI's chances while minimizing player's chances
- Includes depth-based scoring to prefer quicker wins

### Code Structure

**GUI Version (`tictactoe_gui.py`):**
- `TicTacToeGUI` class: Main game controller
  - `setup_menu_screen()`: Creates main menu
  - `setup_game_screen()`: Creates game board
  - `on_cell_click()`: Handles player moves
  - `ai_move()`: Handles AI moves
  - `check_game_end()`: Checks win/draw conditions
  - `highlight_winner()`: Visual feedback for wins

**CLI Version (`tictactoe_cli.py`):**
- `create_board()`: Initializes empty 3×3 board
- `display_board()`: Renders the current board state
- `is_valid_move()`: Validates move legality
- `make_move()`: Places mark on board
- `check_winner()`: Checks for win condition
- `is_board_full()`: Checks for draw condition
- `get_player_move()`: Gets and validates human input
- `play_game()`: Main game loop

**Shared AI Logic (both versions):**
- `get_available_moves()`: Returns list of empty cells
- `minimax()`: Implements minimax algorithm for AI
- `get_ai_move()`: Determines AI move based on difficulty

## 🔧 Troubleshooting

### "ModuleNotFoundError: No module named 'tkinter'"

**Solution:** Install tkinter:
```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# Rocky Linux/Fedora/CentOS
sudo dnf install python3-tkinter

# macOS
brew install python-tk@3.X

# Windows
Reinstall Python with "tcl/tk and IDLE" checked
```

### Sound effects not working

**Note:** Sound effects are system-dependent. They work best on:
- Linux with speakers/audio enabled
- Windows (uses built-in beep)
- macOS (uses system beep)

If you don't hear sounds, try:
1. Check your system volume
2. Ensure audio output is working
3. Toggle sound off/on in Settings to reconfigure

### Statistics not saving

**Solution:** Check file permissions:
```bash
# Ensure write permissions in the directory
chmod 755 ~/path/to/tictactoe/
```

The `tictactoe_stats.json` file should be created automatically in the game directory.

### Window too small or not fitting on screen

**Solution:**
- The window is resizable - drag edges to expand
- Try pressing F11 for fullscreen (if your window manager supports it)
- Reduce text size or board size in Settings

### "Command not found" error

**Solution:** Verify Python installation:
```bash
python3 --version
```

Make sure Python 3.6+ is installed. If not, install:
```bash
# Ubuntu/Debian
sudo apt-get install python3

# Fedora/Rocky Linux
sudo dnf install python3
```

### Keyboard shortcuts not working in game

**Note:** Keyboard shortcuts (1-9) only work in the enhanced version when it's the human player's turn.

In two-player mode or AI's turn, you'll need to click with the mouse.

### Game freezes when playing Hard AI

**Note:** This is normal! Hard AI takes a moment to think (about 1 second). This is done intentionally to make the game feel more playable and less instant. The game is working - just be patient!

**Q: Can I reset my statistics?**
A: Yes! Go to Statistics → Reset Statistics in the Enhanced version.

**Q: Can I change board size mid-game?**
A: No, but you can change it in Settings before starting a new game.

**Q: Is the Hard AI truly unbeatable?**
A: Yes, it uses perfect minimax strategy. Best outcome against Hard: a draw.

**Q: Can I play online?**
A: Not yet - only local multiplayer is supported. Online multiplayer would be a great future feature!
